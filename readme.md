# Brno Public Transport Departure Board

A minimalistic mobile-friendly departure board for public transport in Brno, Czechia. Enter a stop name to see upcoming departures, filter by specific routes, and bookmark your frequent stops.

## Project Structure

```txt
* data
  * GTFS/             # Unzipped GTFS .txt files
  * stops.csv         # Specific Brno stops dataset
  * transit.db        # SQLite database (generated by ingest.py)
* src
  * static/           # Frontend assets (HTML, CSS, JS)
  * ingest.py         # GTFS to SQLite ingestion script
  * app.py            # Flask Backend API
  * requirements.txt  # Python dependencies
```

## Setup & Running

### 1. Prerequisites

- Python 3.12.3+
- `pipenv`

**Windows Warning:**

This project uses the `zoneinfo` module for strict Czechia time handling. If you are running this on **Windows**, you must install the `tzdata` package because Windows does not provide a native IANA timezone database:

```bash
pipenv install tzdata
```

### 2. Installation

```bash
pipenv install
```

### 3. Data Ingestion

Ensure your `data` folder is populated with the sources below. Then run:

```bash
cd src
pipenv run python ingest.py
```

### 4. Run Server

```bash
pipenv run python app.py
```
Access the app at `http://localhost:5000` or your local IP if `RUN_GLOBALLY` is enabled.
